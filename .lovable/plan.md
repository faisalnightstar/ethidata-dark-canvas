

# EthiData & Technologies - Complete Dark Mode Enterprise Website

## Design Foundation

**Color Palette:**
- **Primary Background:** Deep black (#000000 to #0D0D0D)
- **Secondary Background:** Charcoal (#101010 to #1A1A1A)
- **Primary Accent:** Neon Cyan (#00E6FB)
- **Secondary Accent:** Neon Green (#00FF9F)
- **Gradient:** Cyan → Green gradient for key CTAs and highlights
- **Text:** Pure white (#FFFFFF) for headings, muted gray (#A0A0A0) for body

**Visual Identity:**
- Text-based logo "ETHIDATA" with glowing cyan accent
- Abstract geometric tech patterns (circuit-like lines, hexagons)
- Glassmorphism cards with subtle glow effects
- Clean, grid-based layouts with generous spacing

## Technical Approach

- React Router for all 24+ routes
- Reusable component library for cards, buttons, forms, sections
- CSS custom properties for design tokens (easy theming)
- Tailwind utility classes with custom dark theme extensions
- Responsive design (mobile, tablet, desktop breakpoints)
- All placeholder content ready for future replacement
